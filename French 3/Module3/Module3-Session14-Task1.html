Welcome!

This task will help you review the grammar presented in the three modules of this course.



Learning Goals

  • Using the présent simple, the futur simple, the perfect and the imperfect tenses
  • Agreeing names and adjectives in number and gender
  • Replacing nouns with the pronouns le, la, l’, les, y, en
  • Using quantity and intensity adverbs (très, peu, etc.)
  • Using prepositions of place and movement
  • Using the comparative and superlative of adjectives
  • Using adverbs of manner
  • Expressing obligation using Devoir and Falloir
  • Using connectors and conjunctions
