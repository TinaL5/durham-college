GRAMMAR | Reviewing adverbs of intensity and quantity

How to express quantity?

You can use the following expressions to talk about quantities.

No quantity No quantity of something Small quantity Insufficient

Rien

Il n'y a rien dans le frigo.

Aucun + nom

Il n'y a aucun magasin ouvert le dimanche.

Peu de + nom indénombrable

Il y a un peu d’huile dans cette recette.

Quelque + nom dénombrable

Il y a quelques personnes dans le magasin.

Note : quelque is an adjective that agrees in number with the noun it quantifies.

Pas assez de + nom

Il n’y a pas assez de temps.

Pas suffisamment de + nom

Nous n’avons pas suffisamment de temps.

Sufficient All A large quantity Too much

Assez de + nom

Nous avons assez d’eau pour une semaine.

Suffisamment de + nom

Vous avez suffisamment d’épices.

Tout + nom

Marc achète toutes les courses.

Beaucoup de + nom

Sarah a beaucoup de recettes de cuisine.

Trop de + nom

Il boit trop de café.

How to express intensity?

You can use the following adverbs to express intensity. They are placed before the adjective.

Very Too

Très + adjective

Cette recette est très bonne.

Trop + adjective

Ce plat est trop chaud.
