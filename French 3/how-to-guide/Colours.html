Colours

This page demonstrates different options for colours. Using thes colours will help ensure consistency across courses.

If you decide to change colours, please ensure that your new colours are accessible by using the HTML Editor Accessibility checker. It can be found in the lower right hand corner of the HTML Editor .

Colours

If you're using one of the course template pages, the colours will automatically be applied. This page gives you the option to apply other colours as you'd like while ensuring the colours are consistent with the template. Just copy the HEX code for the colour you wish to use (in bold below) and paste it into the colour selector in the HTML editor.

This video will walk you through the steps involved to copy these colours into your pages.
