Conversation dans un train

Vous êtes assis dans le train pour vous rendre à Québec. Le voyage est long. Un jeune homme vous demande la permission de lire avec vous le journal que vous lisez. Vous acceptez.

Complétez le dialogue pour répondre au jeune homme. Utilisez les phrases suivantes:

Oui, bien sûr / je vais bien merci / De rien / je peux vous donner la page sportive / aimer + infinitif / mais
